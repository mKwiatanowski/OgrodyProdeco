# Dokumentacja źródłowa – ogrodyprodeco.pl

## Metodologia i ograniczenia

Zgodnie z prośbą, próbowano pobrać zawartość ogrodyprodeco.pl z zasobów historycznych (Wayback Machine / archive.org). Napotkano dwa ograniczenia techniczne w tej sesji:

1. **archive.org / web.archive.org jest zablokowane** dla narzędzi do pobierania treści (blocklist na poziomie środowiska) — nie udało się pobrać zrzutów historycznych z Wayback Machine.
2. **Aktualna strona ogrodyprodeco.pl jest aplikacją renderowaną w całości po stronie klienta (SPA)** — pobranie surowego HTML nie zwraca żadnej treści tekstowej bez wykonania JavaScript, a przeglądarka (Claude in Chrome) była w tej sesji niedostępna (rozłączone rozszerzenie).

**Co zrobiono zamiast tego:** treść firmy odtworzono na podstawie:
- wyników wyszukiwania wskazujących adresy podstron obecnej strony (strona-glowna, o-nas, projekt, kontakt),
- profilu firmy w katalogu branżowym Oferteo (zawiera opis "O nas" prawdopodobnie skopiowany przez właściciela z własnej strony, dane adresowe, NIP, usługi, opisy zdjęć realizacji),
- innych katalogów branżowych (PlanujemyOgrod.pl, ALEO.com, Panorama Firm) potwierdzających te same dane firmowe.

Poniższe pliki to **zrekonstruowane, opisowe streszczenie** rzeczywistej zawartości firmy (nie dosłowny zrzut HTML), pogrupowane wg prawdopodobnej struktury podstron oryginalnej witryny. Źródła podane przy każdym pliku.
